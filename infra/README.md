Infra
=====

Dieser Ordner enthält Infrastruktur- und Deployment-bezogene Dateien.

Inhalte
-------
- `docker-compose.yml`: Lokale Entwicklungsumgebung für Backend (FastAPI) und Frontend (Next.js).

Nutzung
------
- Kopiere `.env.example` zu `.env` in der Projektwurzel.
- Starte Dienste:
  - `cd infra && docker compose up`
